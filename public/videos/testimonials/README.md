# Testimonial Videos

Real solo-sailor stories. One file per sailor.

Naming: `<sailor-name>.mp4` + `<sailor-name>-poster.jpg` (e.g. `deeann.mp4`, `deeann-poster.jpg`).

Specs: 1920×1080 H.264 MP4 **with audio**, 30–60 s, target 10–20 MB per file. Get a **written release** from each sailor before adding their clip; keep the signed release out of the repo (add `release.pdf` etc. to `.gitignore`).

See master `../README.md`.
