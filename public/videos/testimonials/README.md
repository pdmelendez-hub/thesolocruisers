# Testimonial Videos

Real solo-sailor stories. Used on `/` (testimonials section), `/destinations`, and social.

**Structure:** create one subfolder per sailor (lowercase, dash-separated).

```
testimonials/
├── deeann-caribbean-2025/
│   ├── deeann.mp4
│   └── deeann-poster.jpg
├── jason-mediterranean-2024/
│   ├── jason.mp4
│   └── jason-poster.jpg
└── ...
```

Specs: 1920×1080 H.264 MP4 **with audio**, 30–60 s, target 10–20 MB per file. Always include a `*-poster.jpg` still.

Get **written release** from each sailor before adding their clip — keep the signed release alongside the video (e.g. `release.pdf`) but **add `release.pdf` to .gitignore** so it doesn't deploy.
