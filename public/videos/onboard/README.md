# Onboard B-Roll

Ship and onboard experience clips. Used for inline page sections, social posts, and storytelling.

| Subfolder | What goes here |
|---|---|
| `scarlet-night/` | Virgin Voyages' fleet-wide party night |
| `the-galley/` | Communal-dining-style food hall clips |
| `deck-spaces/` | Pools, sundecks, sunset shots from the deck |
| `solo-cabins/` | Cabin walkthroughs, Sea Terrace solo, NCL Studios |
| `group-mixers/` | TSC-hosted events, cocktail mixers, theme nights |

Naming: `<topic>-01.mp4`, `<topic>-02.mp4`, … e.g. `pool-deck-01.mp4`.

Specs in master README at `/public/videos/README.md`. **Stay under 24 MB per file.**
