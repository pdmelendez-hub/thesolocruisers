# Destination B-Roll

Atmospheric port/region clips. Used on `/destinations` page, in cruise card hover/expand panels, and in marketing.

| Subfolder | What goes here |
|---|---|
| `alaska/` | Glaciers, Inside Passage, Juneau, Skagway, whales |
| `caribbean/` | Turquoise water, beaches, Nassau, Cozumel, St. Thomas |
| `mediterranean/` | Santorini, Barcelona, Amalfi, port arrivals |
| `mexican-riviera/` | Cabo arch, Mazatlán, Puerto Vallarta |
| `panama-canal/` | Lock transits, jungle, two-oceans imagery |
| `bermuda/` | Pink sand, pastel colonial streets |

Naming: `<port-or-region>-01.mp4`, e.g. `juneau-glacier-01.mp4`, `cabo-arch-sunset-01.mp4`.

Specs in master README. **Stay under 24 MB per file.**
