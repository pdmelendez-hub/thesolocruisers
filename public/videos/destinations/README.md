# Destination B-Roll

Atmospheric port/region clips. Used on the `/destinations` page, as section accents, and in marketing.

Naming: `destinations-<topic>.mp4` (e.g. `destinations-philippines-lagoon.mp4`, `destinations-venice-gondola.mp4`).

See `../INVENTORY.md` for the current list. Specs in the master README at `../README.md`. **Stay under 24 MB per file.**
