# Hero Assets — Destinations (`/destinations`)

Drop in files below; the page picks them up automatically. All optional —
falls back to a Pexels image via `/api/photo` if absent.

| File | Used in | Notes |
|---|---|---|
| `main-video.mp4`  | Page hero | Looping b-roll behind the "Destinations" headline |
| `main-video.webm` | Page hero | Optional smaller WebM |
| `main-poster.jpg` | Page hero | Still / fallback |

## Recommended specs
- Video: 1920×1080, H.264 MP4 (+ optional WebM), 8–15 s, silent loop, < 8 MB.
- Poster: 1920×1080 JPG/WebP, < 300 KB.

Pick a wide, atmospheric shot of a port/coastline at golden hour to match the
OTGG family look.
